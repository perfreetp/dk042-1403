## 1. 产品概述

校车救援指挥台是面向校车运营公司调度员的Web端应急处置系统，专注于"车辆故障后如何最快安排替代车辆和维修人员"的核心场景。系统通过标准化的故障接报流程、智能化的资源匹配和可视化的处置跟踪，帮助调度员在紧急情况下快速决策、高效调度，最大限度降低学生滞留风险和道路安全隐患。

## 2. 核心功能

### 2.1 用户角色

| 角色 | 登录方式 | 核心权限 |
|------|----------|----------|
| 调度员 | 账号密码登录 | 故障接报、资源调度、处置跟踪、历史记录查询 |

### 2.2 功能模块

1. **故障接报页**：故障信息录入、风险等级评估、快速接报
2. **资源匹配页**：附近资源展示、方案选择、调度单生成与通知
3. **处置跟踪页**：时间轴进度展示、超时提醒、节点确认

### 2.3 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|----------|----------|----------|
| 故障接报页 | 故障信息表单 | 录入车牌、线路、当前位置、学生人数、故障现象、是否占道 |
| 故障接报页 | 风险评估面板 | 实时计算并显示学生滞留风险、道路风险、资源距离的红黄绿三色提示 |
| 故障接报页 | 快速操作区 | 一键提交接报、保存草稿、取消 |
| 资源匹配页 | 资源列表 | 展示附近空闲校车、跟车照管员、维修点、拖车联系人，按距离排序 |
| 资源匹配页 | 方案选择 | 支持选择一套完整救援方案（接驳车+维修+拖车） |
| 资源匹配页 | 调度单生成 | 生成调度单，分别通知接驳司机、维修人员、学校值班老师 |
| 处置跟踪页 | 时间轴 | 按时间顺序展示各处置节点（已接单、已出发、到达现场、学生转运完成、故障车拖离） |
| 处置跟踪页 | 超时提醒 | 节点超时未反馈时高亮提醒调度员确认 |
| 处置跟踪页 | 操作按钮 | 支持手动确认节点、添加备注 |

## 3. 核心流程

调度员接到司机故障电话 → 录入故障信息 → 系统自动评估风险等级 → 查看附近可用资源 → 选择救援方案 → 生成调度单并通知各方 → 实时跟踪处置进度 → 超时节点提醒确认 → 救援完成归档

```mermaid
    D --> E["选择救援方案"]
    E --> F["生成调度单"]
    F --> G["通知各方<br/>(司机/维修/学校)"]
    G --> H["处置跟踪时间轴"]
    H --> I["超时提醒确认"]
    I --> J["救援完成"]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深空蓝（#0a1628）作为背景，营造专业指挥中心氛围
- **强调色**：
  - 红色（#ef4444）：高风险/紧急
  - 黄色（#f59e0b）：中风险/注意
  - 绿色（#10b981）：低风险/正常
  - 蓝色（#3b82f6）：信息/操作
- **字体**：使用 JetBrains Mono 等宽字体作为数字显示，Inter 作为正文，增强科技感和可读性
- **布局风格**：卡片式布局，深色主题，适合长时间值班使用
- **图标风格**：线性图标，与深色背景形成对比

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|----------|----------|----------|
| 故障接报页 | 顶部导航 | 系统标题、当前时间、调度员信息、页面切换标签 |
| 故障接报页 | 左侧表单区 | 表单输入框、下拉选择、开关控件、表单验证提示 |
| 故障接报页 | 右侧风险面板 | 三色风险指标卡片、风险说明文字、脉冲动画效果 |
| 故障接报页 | 底部操作栏 | 提交按钮（主按钮）、保存草稿按钮（次按钮） |
| 资源匹配页 | 资源分类标签 | 校车/照管员/维修点/拖车 分类切换 |
| 资源匹配页 | 资源卡片列表 | 距离标识、状态标签、联系方式、选择按钮 |
| 资源匹配页 | 已选方案侧栏 | 已选资源汇总、调度单预览、确认生成按钮 |
| 处置跟踪页 | 故障概览卡片 | 车牌、位置、学生人数、故障时间等关键信息 |
| 处置跟踪页 | 时间轴组件 | 垂直线性时间轴、节点状态图标、时间戳、操作按钮 |
| 处置跟踪页 | 超时提醒条 | 顶部悬浮提醒，显示超时节点列表 |

### 4.3 响应式

- 桌面端优先设计，最低适配 1366×768 分辨率
- 主要针对运营中心大屏幕显示器优化
- 支持中等屏幕自适应布局

### 4.4 动效与交互

- 风险等级变化时的颜色过渡动画
- 高风险状态的脉冲呼吸效果
- 时间轴节点更新的滑入动画
- 按钮悬停的微交互效果
- 页面切换的平滑过渡
